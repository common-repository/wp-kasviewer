== Description ==
KasViewer is a image, or web content visor, base on sexlightbox script to create a popup box in the same frame to show html content.

This plugins is, in other words, Sexy Lightbox for mootools 1.2.3 to wordpress.

== Installation ==

Wordpress basic instalation.

Option 1
 
1. Download.
2. Unzip. 
3. Upload the wp-kasviewer folder to the plugins directory.
4. Activate the plugin.
5. Go to new KasPlugins menu.
6. Select KasViewer, and set personal settings.

Option 2
 
1. Go to addnew plugins on Plugins menu.
2. Find wp-kasview. 
3. Press install.
4. Later, Activate the plugin.
5. Go to new KasPlugins menu.
6. Select KasViewer, and set personal settings.

= Upgrade =

1. Press upgrade plugin in wordpress plugins list when plugins have a new versión.
